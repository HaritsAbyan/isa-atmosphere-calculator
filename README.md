# ISA Atmosphere Calculator

A Python-based tool for calculating International Standard Atmosphere (ISA) parameters. 
This project is part of my preparation for Aerospace Engineering studies.

## Features
- **Layer Detection:** Automatically switches logic between Troposphere and Stratosphere.
- **Precision Metrics:** Calculates Pressure, Density Altitude, Air Density, and Sonic Speed.
- **Advanced Physics:** Implements Sutherland's Law for Dynamic Viscosity.

## How to Use
1. Clone this repository.
2. Run `python atm_calc.py` in your terminal.
3. Input altitude (meters) and sea-level temperature (Celsius).
